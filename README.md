## Smart Power Meter with Reinforcement Learning

This project implements an intelligent power monitoring and energy optimization system using Reinforcement Learning (RL). The system simulates real-time electricity consumption, stores readings in a database, predicts future usage trends, and optimizes energy consumption decisions.
